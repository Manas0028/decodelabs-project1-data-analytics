# Data Cleaning & Preparation -- DecodeLabs Project 1

## 📌 Project Overview

This project was completed as part of the DecodeLabs Data Analytics
Internship Program.

The objective of this project is to clean and prepare a raw e-commerce
dataset by identifying missing values, validating data quality, checking
for duplicates, and ensuring data consistency.

## 🎯 Project Objectives

-   Identify and handle missing values
-   Detect and remove duplicate records
-   Validate date and numeric formats
-   Prepare an analysis-ready dataset

## 🛠 Tools & Technologies Used

-   Python
-   Pandas
-   Microsoft Excel
-   Jupyter Notebook

## 🔍 Findings

-   309 missing values in CouponCode
-   No duplicate rows
-   No duplicate Order IDs
-   Valid date formats
-   Accurate TotalPrice calculations

## 🧹 Data Cleaning Steps

1.  Identified missing values.
2.  Replaced missing CouponCode values with "No Coupon".
3.  Checked for duplicates.
4.  Validated date formats.
5.  Verified TotalPrice calculations.

## ✅ Results

The dataset is clean, consistent, and ready for analysis.
